# kyverno-http-authorizer

Authenticate and authorize HTTP requests using Kyverno policies

## Overview 

This HTTP authorization server provides policy-based access control for HTTP requests using Kyverno policies. It can be integrated with various proxies, gateways, and service meshes that support external authorization services to make authorization decisions based on Kyverno policies. 

**WARNING: ⚠️ Kyverno HTTP Authorizer is in development stage.**
